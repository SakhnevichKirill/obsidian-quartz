# Расширенный аудит брендинга и SWOT по языку ценностей аватаров

Дата: 2026-03-04  
Проект: Чеховский / лендинг кейтеринга  
Сравнение: бренд-платформа `./branding` + текущие проектные артефакты + конкурент `ik-catering.ru`.

## 1. Диагноз

### 1.1 Что уже сильно в бренд-платформе Чеховского

1. Есть четкое брендовое ядро: «интеллигентная кулинария по соседству», «горячо вовремя», «с любовью».  
2. Есть готовые смысловые платформы, которые можно капитализировать в digital:
- `Слово Чеховского` (гарантия времени/температуры),
- `Штамп заботы` (доказательство аккуратности/свежести),
- `Вечерние рассказы` (готовые решения на ужин),
- `Легкий жанр` (ПП-сигналы).
3. В документах уже задан ToV и библиотека фраз, то есть проблема не в отсутствии контента, а в переносе в продуктовые сценарии.

### 1.2 Критический разрыв между брендингом и текущими наработками по лендингу

Главный разрыв: бренд-платформа описывает в первую очередь формат «кулинария у дома/ритейл», а текущий лендинг v4 решает задачу «кейтеринг/фуршет с переводом в корзину старого сайта».

Фактически:
- В brand-доках почти нет языка «кейтеринг/фуршет» (кроме единичного упоминания в ТЗ части 3).
- В `TZ v4` почти нет брендовых маркеров `Слово Чеховского`, `Штамп заботы`, `Легкий жанр`, хотя они могут усиливать доверие и отстройку.
- Аватары в brand-платформе и в `TZ v4` пересекаются частично, но описаны разными языками выгод.

### 1.3 Что у конкурента по языку ценностей (IK-Catering)

Конкурент говорит простым транзакционным языком:
- понятный оффер месяца с ценой,
- четкие условия доставки и минималки,
- обещание скорости ответа,
- «заказать» и корзина как прямой action.

Это не «поэтичный» язык, но он быстро снимает неопределенность перед заказом.

Вывод: Чеховскому нужно сохранить эмоциональную литературную ДНК, но сделать функциональную часть такой же прозрачной и короткой, как у конкурента.

## 3. SWOT по брендингу и языку ценностей

| Блок | Содержание |
|---|---|
| **S (Strengths)** | 1) У бренда есть уникальный язык и ритуалы, которых нет у «безликих» каталогов. 2) Сильная эмоциональная платформа «вовремя + забота + как дома». 3) Готовый набор фраз/ToV/визуальных принципов для масштабирования. |
| **W (Weaknesses)** | 1) Неполная связка бренд-платформы с кейтеринг-сценарием лендинга. 2) B2B-персона в бренд-платформе не доописана. 3) Ценностный язык для аватаров не формализован как «проблема -> обещание -> доказательство -> CTA». |
| **O (Opportunities)** | 1) Собрать гибрид: эмоциональная отстройка Чеховского + транзакционная ясность конкурента. 2) Ускорить конверсию на лендинге за счет branded-proof блоков в карточках. 3) Сделать единый словарь ценностей для sales, рекламы, лендинга и колл-скриптов. |
| **T (Threats)** | 1) Если оставить только «поэзию», проиграем быстрым прагматичным офферам. 2) Если оставить только «перформанс-язык», потеряем брендовую маржу и лояльность. 3) Без единого словаря аватаров разные каналы будут говорить разными голосами. |

## 4. Аудит языка ценностей по аватарам

## 4.1 Единая матрица «аватар -> ценность -> язык -> доказательство"

| Аватар | Главная задача клиента | Язык ценности (что обещаем) | Доказательство | CTA-формула |
|---|---|---|---|---|
| 1. Организатор семейного события | Накрыть стол без стресса и провалов | «Празднично, вовремя, без лишних хлопот» | `Слово Чеховского` + топ-наборы + понятная цена | «Подобрать набор за 2 минуты» |
| 2. Офис-менеджер / HR | Быстро собрать решение под бюджет и число гостей | «Предсказуемо по срокам и бюджету» | SLA ответа + структура наборов + прозрачные условия доставки | «Получить расчет и перейти к оформлению» |
| 3. Ассистент руководителя | Не ошибиться перед руководителем/гостями | «Статусно подано, аккуратно исполнено» | `Штамп заботы` + качество подачи + кейсы/фото | «Уточнить сценарий по звонку» |
| 4. Осознанный едок (из бренда) | Нужен ПП-выбор без долгого поиска | «Легкий выбор без компромисса вкуса» | `Легкий жанр`, КБЖУ, фильтр ПП | «Показать ПП-позиции» |
| 5. Сладкоежка/подарок (из бренда) | Закрыть эмоцию и комплимент в заказе | «Десерт как финальный штрих события» | «Эпилог» коммуникации + подарочные визуалы | «Добавить десерт к заказу» |

### 4.2 Критичные gaps по аватарам в текущем v4

1. В `v4` хорошо покрыты 3 ивент-аватара, но не встроены брендовые механики доверия (`Слово`, `Штамп`) в их путь.
2. ПП и десертный язык из бренд-платформы не доведен до сценариев карточек/секций (а это прямой потенциал роста среднего чека и выбора).
3. Для B2B нет отдельного «языка снижения риска» (срок, ответственность, сценарий согласования), хотя это один из ключевых барьеров покупки.

## 5. Что конкретно добавить в наши наработки

### 5.1 В `catering-landing-tz-v4-assortment-first.md`

1. В Hero добавить под оффером короткий бренд-proof блок:
- `Слово Чеховского: горячо и вовремя`,
- `Штамп заботы: аккуратная подача и контроль`,
- `ПП-метки: легкий выбор без лишнего поиска`.

2. В expanded-card добавить поле `Почему можно доверять` (1 строка с proof-маркером).

3. Для каждого аватара зафиксировать «главный барьер -> контрфраза -> CTA» как обязательный копирайт-блок.

4. Добавить отдельный KPI по бренд-сигналам:
- `brand_proof_view_rate`,
- `brand_proof_assisted_outbound_rate`.

### 5.2 В операционный контур контента

1. Вынести «словарь ценностей» в отдельный мини-стайлгайд для всех каналов (лендинг, колл, мессенджер, карточки).
2. Разделить формулировки на 2 режима:
- `Functional`: быстро/понятно/по цене/по сроку,
- `Chehovskiy`: тепло/достойно/с заботой.
3. В каждом ключевом блоке использовать обе линии одновременно (не выбирать только одну).

## 6. SMART-гипотезы по брендингу (поверх текущего CRO)

### H1. Бренд-proof в hero снижает тревожность первого шага

- **Specific**: добавить 3 proof-маркера в hero.
- **Measurable**: `hero_cta_click_rate +15%` к базовой неделе.
- **Achievable**: у конкурента работают простые «снятия неопределенности», у нас будет усиление брендом.
- **Relevant**: влияет на старт воронки.
- **Time-bound**: 14 дней после выката.

### H2. Proof в карточках увеличивает переход в старый сайт

- **Specific**: в каждой expanded-card показывать proof-бейдж + CTA.
- **Measurable**: `sku_legacy_click_ctr >= 11%` (против цели 10% в v4).
- **Achievable**: минимальное изменение интерфейса, сильный смысловой эффект.
- **Relevant**: прямой вклад в лидогенерацию.
- **Time-bound**: 30 дней.

### H3. Аватарный язык увеличивает конверсию «раскрытие -> действие"

- **Specific**: сделать 3 варианта copy для 3 аватаров в ключевых секциях.
- **Measurable**: рост `expand_to_action_rate` на `>=12%`.
- **Achievable**: база фраз уже есть в бренд-платформе.
- **Relevant**: связывает бренд и performance.
- **Time-bound**: 30 дней A/B.
